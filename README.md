## Exercice 1 : REVERSIBILITE DE NOMBRES (NOMBRES PALINDROMES)

### Solution :
Le programme définit une fonction `palindromes_numeriques` qui parcourt chaque nombre de la liste donnée,
convertit le nombre en chaîne et vérifie si la chaîne est identique à son inverse.
Les nombres qui respectent cette condition sont ajoutés à une liste de résultats, qui est ensuite renvoyée.

### Exemple :
Voici une liste d'entrée : [121, 123, 454, 12, 7, 88, 1001]  
Nous aurons comme liste de sortie : [121, 454, 7, 88, 1001]

## Exercice 3 : CONVERTISSEUR DE DATE

### Solution :
Le programme définit une fonction `convertir_date` qui :
- Prend une date au format "AAAA-MM-JJ".
- Vérifie la validité de la date avec la fonction `date_valide`.
- Convertit la date valide au format "JJ/MM/AAAA".
- Gère les années bissextiles pour février.
- Retourne un message d'erreur si la date est invalide sinon retourne la date au format "JJ/MM/AAAA".

### Exemple :
- Date en entrée : "2022-12-16"
- Date de sortie : "16/12/2022"

# EXERCICE 5 : COMPRESSION DE TEXTE

### Solution :
Le programme définit une fonction `compression_texte` qui :
- Parcourt chaque caractère de la chaîne.
- Compte les répétitions consécutives d'une lettre.
- Ajoute la lettre suivie de son compteur à une nouvelle chaîne.
- Retourne la chaîne compressée.

### Exemple :
- Texte a l'entrée : "aaaaabbbcc"
- Texte de sortie : "a5b3c2"
